Raspberry Pi HAT

Power & Interface HAT

Project Overview

Students had a need for a simple way to power a Raspberry Pi for their various academic projects, so I designed this "starter HAT" (Hardware Attached on Top) to fill that need. It features a buck regulator capable of outputting 4A and a blank canvas so student can modify the board files to meet their project needs.

Skills & Learning

I wanted a robust voltage regulator layout that could withstand any amount of abuse a student could throw at it, so I carefully read the datasheet's application notes to ensure proper implementation and reliability. Like many of my first revisions, it was not perfect the first time. To those that are interested, I did not correctly connect the analog ground of the regulator to the system ground, so the board did not operate. This was fixed with a jumper wire and will be corrected soon in the next revision.

This was another great project to strengthen my KiCad skills and ability to extract important information from component datasheets.
