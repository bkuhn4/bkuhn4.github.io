PCB Clock

ESP32 IoT Clock

Project Overview

For this personal project, I challenged myself to create an ESP32-based board with sensor integration, IoT capability, and a fun user-experience. This was one of my first PCBs, so the objective was to learn several different aspects of PCB design and prepare a demonstration of my skills to present to employers.

Key Features

  • 7 segment displays and 60 LEDs to display time
  • WiFi connectivity to sync time
  • Light and temperature sensors
  • Piezo buzzer to set alarms and timers
  • USB-C power and programming
  • An interactive arcade-style game

Skills & Learning

Like many of my projects, the board was made in KiCad and I used reference component data sheets and other similar projects in the design. This project greatly strengthened my hardware design and embedded programming skills. There are several issues with this first revision, but I've debugged them and learned from those mistakes.
